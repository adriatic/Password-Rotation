# Password Rotation Project

This project contains scaffolding for a macOS Zsh-based password-rotation utility.

## Project Structure
Refer to the "Project Structure" section in Project-Definition.md

#### Note: in order to create the initial content if this repository you need to go threough the following steps
1. Create empty repository at "https://github.com/adriatic/Password-Rotation"
2. Clone this repository to "/Users/nik/Work/AI/Apps". This step will create the local .git file referencing the "https://github.com/adriatic/Password-Rotation" as it's remote.
3. Run the local command "sh init_project.sh" which creates the initial project locally and pushes it to remote

The "init_project.sh" script understands the relationship betwwen the local and remote repo. This file is added to the repo after initial creation.
